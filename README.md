# DataPulse Lab

Enterprise-style service company website for DataPulse Lab, built with Next.js,
React, Tailwind CSS, Framer Motion, and lucide-react.

## Pages

- `/`
- `/services`
- `/services/[slug]`
- `/projects`
- `/projects/[slug]`
- `/industries`
- `/insights`
- `/about`
- `/contact`
- `/privacy`
- `/terms`

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
